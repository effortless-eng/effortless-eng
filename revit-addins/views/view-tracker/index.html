Have you ever had so many views in so many files open that even colorized tabs couldn’t help you? (kudos to Ehsan for pyRevit 😉) Unfortunately there is a limit where horizontal lists like the view tabs get unmanageable, and you loose track of what you have open. The other built-in lists which are provided are quite inconvenient, and difficult to overview.

Then comes View Tracker to save your day!

This tool provides an user friendly way of managing your currently or recently open views. Your views will be organized in a simple dockable panel, grouped by their parent file. You are also able to collapse these groups to free up real-estate, and you can close any view or project if you wish to. You closed a view by accident? Don’t worry, we got you covered! Just enable Show Recent Views and you can see the views which were open in your current session.

This tool can be extremely helpful for BIM Managers, or any powerusers who have to deal with many files and many views simultaneously.

Layout #

In the dockable panel you can see all your currently open views. They are grouped by their parent file, which groups can be collapsed. The views are sorted by their type and by their name. Their type is shown by the little icon on the left, and also in the tooltip, above their view template if there is any. Your active view is shown in bold. Your views will be automatically added once opened.

Furthermore the actual count of open views will be shown in the group header on the right in the following format: file / total.

Functions #

Activating a view #

Quite straightforward, just double-click on any, even if it is closed already, but still tracked as a recent view.

Collapse a group #

Clicking on any group header will collapse or expand it’s content accordingly.

Closing a view / remove from recents #

Middle mouse button clicks will close any open views, can be used to remove any recent views too.

Context menu #

Using the right mouse button you can get the context menu which has the following functions:

  • Close View
    Closes the highlighted view

  • Close File
    Closes the file which contains the highlighted view

  • Show Recent Views
    Toggles the visibility of recent views

  • Clear Recent Views
    Removes all recent views from the panel
